1. Add the SQL Server JDBC Driver

    Download the Microsoft JDBC Driver for SQL Server (mssql-jdbc-<version>.jar).
    Place the JAR in the WILDFLY_HOME/modules/com/microsoft/sqlserver/main/ directory.
    Create a module.xml in the same directory:

standalone.xml/domain.xml
<driver name="sqlserver" module="com.microsoft.sqlserver">
    <xa-datasource-class>com.microsoft.sqlserver.jdbc.SQLServerXADataSource</xa-datasource-class>
</driver>
