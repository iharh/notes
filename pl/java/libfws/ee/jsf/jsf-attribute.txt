We briefly describe the f:attribute, f:param, and f:facet tags here. 
Any component can store artibrary name/value pairs in its attribute map. 
You can set an attribute in a page and later retrieve it programatically. 
For example, in “Supplying Attributes to Converters” on page 260 of Chapter 6, 
we set the separator character for credit card digit groups like this:

<h:outputText value = "#{payment.card}">
    <f:attribute name = "separator" vlude = "-" />
</h:outputText>
