------------------------------------------------
xdb与hdb的区别？
What’s the difference between xdb and hdb?
 
hdb是跨平台跨机器的，和机器字节序无关，且每条记录比xdb节省4字节空间
hdb can work under cross-platform and cross machine environment, it has nothing to do with machine’s byte order, and it saves 4 bytes memory space per record comparing with xdb
xdb为兼容C版的xdb而编写，和机器字节序有关，其余均差不多.
xdb is written in the purpose of being compatible with the C version xdb, it is related to machine byte order. No more difference
