    Override method

    SmartType completion

    New File
C-/
    Comment out code block with //
C-S-/
    Comment out code block with /*...*/

    list of valid params

    Code Surround With (for, while)

    Go to Declaration

    Navigate|Symbol

    Navigate to prev/next method

    Navigate to prev/next compiler error
