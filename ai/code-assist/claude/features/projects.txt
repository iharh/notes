Anthropicовцы добавили в Claude Cowork фичу Projects. 
Суть простая: один проект = одно место для всего. Файлы, задачи, инструкции — всё рядом, и Claude помнит контекст между сессиями.

Больше не надо в каждом чате заново рассказывать «я тут делаю проект, вот структура, вот требования». Закрыл — открыл — Claude в курсе.

Можно импортнуть готовый проект или собрать новый. Данные хранятся локально, никуда не улетают.
