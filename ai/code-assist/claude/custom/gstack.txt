Суть: вместо одного ассистента, который делает всё одинаково средне, ты вызываешь нужного специалиста:

* /plan-ceo-review — «а ты вообще правильную штуку строишь?» Ищет 10-звёздочную версию твоей идеи
* /review — параноидальный code review, ловит баги, которые проходят CI
* /browse — агент открывает браузер, логинится, кликает по страницам, делает скриншоты. QA за 60 секунд
* /ship — тесты, пуш, PR одной командой
* /qa — анализирует дифф, сам находит затронутые страницы и тестирует

Рабочий процесс Тана: сначала /plan-ceo-review → потом /plan-eng-review → код → /review → /qa → /ship. Полный цикл через slash-команды.

Особенно нравится /browse — агент наконец-то не слепой и может сам посмотреть, что натворил.
