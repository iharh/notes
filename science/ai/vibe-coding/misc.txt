▪️ Cursor (http://cursor.com/) — редактор кода, форк от VSCode. $20 в месяц, 14 дней триала. Дается N быстрых запросов, по истечению, будут медленные. Точно стоит своих денег, пока не попробуешь — не поймёшь. Текущая проблема (март 2025) — большинство пользователей сидят на актуальном Claude Sonnet, а у Anthropic (владеют Claude) не могут предоставить достаточное количество серверов, в итоге когда в США наступает рабочий день, то у всех пользователей Claude (и Cursor и часто другие продукты) начинаются проблемы с доступом, поэтому приходится использовать другие модели.
 
✔️ Отзывы об инструменте:
Пост (https://vas3k.blog/notes/vibe_coding/) Вастрика
 
▪️ Windsurf (https://codeium.com/windsurf) — конкурент Cursor, стоит от $15 в месяц, тут своя система кредитов и по отзывам, эти кредиты в подписке улетают намного быстрее чем у Cursor и приходится платить pay-as-you-go больше чем за Cursor
 
▪️ Trae (https://www.trae.ai/) — ещё одна IDE для вайб-кодинга от Byte Dance (Китай). Бесплатно с Claude Sonnet 3.7 (имейте ввиду что в бесплатном продукте ваши данные могут быть скормлены аналитике)
 
▪️ Zed (https://zed.dev/) — минималистичный редактор на Rust с GPU-accelerated интерфейсом, после логина через github дает бесплатные лимиты sonnet 3.5, 3.7 и очень быстрый комплишн по табу на базе собственной модели Zeta (файнтюн Qwen2.5-Coder-7B)
 
▪️ Aider (https://aider.chat/) — CLI инструмент, AI-помощник для кодинга, поддерживающий голосовое кодирование (с определенными моделями). Для работы использует две модели в ролях Architect (модель постарше) и Editor (модель помладше). Он бесплатный, нужно только принести свои ключи от моделей.
 
▪️ Roo Code (https://github.com/RooVetGit/Roo-Code) — AI-ассистент для кодинга, интегрированный в VS Code. Предоставляет функции автоматизации, поддержку нескольких моделей и экспериментальные возможности. Бесплатен, требуется предоставить свои API-ключи для моделей.
 
▪️ Claude Code (https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) — CLI инструмент от Anthropic, поддерживает только модели Claude, бесплатный, но нужно оплачивать за API запросы к моделям. По ощущениям, стоит очень дорого, но работает оч хорошо. Примерно стоит $20 за один рабочий день.
 
▪️ Cline (https://github.com/cline/cline) — CLI тул, AI—помощник. Бесплатный, нужно только принести свои ключи
 
▪️ v0 (http://v0.dev/) — Платформа для генерации UI-компонентов и создания приложений с интеграцией Next.js. Поддерживает импорт из Figma и упрощенное развертывание. Бесплатный тариф: до 10 генераций UI-компонентов в день. Premium: $20/месяц — увеличенные лимиты на генерацию и безлимитные проекты. Team: $30/пользователь в месяц — совместная работа и централизованное управление.
 
▪️ Bolt (http://bolt.new/) и open-source (https://bolt.diy/) версия — AI-платформа для создания приложений с акцентом на хранение данных и развертывание. Подходит для начинающих благодаря щедрому бесплатному тарифу. Бесплатный тариф: до 500 МБ хранения, ограниченные AI-запросы (10–20 в день). Платные тарифы от $20/месяц
 
✔️ Отзывы об инструменте:
Отзыв (https://pixeljets.com/blog/lovable-dev-vs-bolt-new/) клубня (en)
 
▪️ Lovable (http://loveable.dev/)  — Платформа для создания приложений без технических навыков. Просто опишите идею, и инструмент создаст приложение. Бесплатный: до 5 кредитов в день, платные от $20/месяц
 
▪️ Replit (http://replit.com/) — Облачная IDE с поддержкой более чем 50 языков программирования. Включает AI-помощника Replit Agent для автозавершения, отладки и рефакторинга кода. Бесплатный план: ограниченный доступ к AI Agent. Платные тарифы от $15/месяц
 
▪️ VS Code Copilot Edits (https://code.visualstudio.com/docs/copilot/copilot-edits) — тот самый Copilot, плагин для редактирования кода прямо в VS Code через Copilot Chat с режимом Agent. Входит в подписку GitHub Copilot.
