У Васи есть специальный инструмент - ножницы для разрезания математических графов.
Этот инструмент может разрушать любое выбранное мальчиком из оставшихся ребер в графе.
Играясь с этим инструментом, мальчик каждый раз задается вопросом, связаны ли какие то две вершины в графе, или Вася их уже разделил.
Известно, что мальчик разрезает граф, пока есть неразрезанные ребра.
Это означает, что к концу игры у мальчика есть граф, в котором нету ребер.

Ввод
В первой строке задано целые числа N M K (0 <= N <= 50000, 0 <= M <= 100000, 0 <= K <= 150000) — количество вершин и ребер в исходном графе, а также число запросов.
В следующих M строках заданы пары чисел U V (1 <= U,V <= N) - ребра графа, связывающие вершины под номерами U и V, нет случаев, когда U равно V.
Далее идет K строк с запросами двух типов:
    cut L R - удалить ребро между вершинами L и R
    ask L R- Вася просит ответить, находятся ли вершины L и R в одной компоненте

Вывод
Выведите "YES" или "NO" без кавычек в одной строке для каждого запроса ask,
"YES" - если вершины в одной компоненте, и "NO" если в разных

Пример ввода:
3 3 7
1 2
2 3
3 1
ask 3 3
cut 1 2
ask 1 2
cut 1 3
ask 2 1
cut 2 3
ask 3 1

Пример вывода:
YES
YES
NO
NO
