У Мислава есть N стаканов бесконечного объёма и каждый стакан одержит некоторое количество воды.
Мислав хочет выпить всю воду, но он не хочет пить более чем из K стаканов.
Мислав может переливать воду из одного стакана в другой.

К несчастью, существенно, какой стакан выберет Миcлав, поскольку они находятся на разном расстоянии от него.
Более точно, количество усилий, которое требуется, чтобы перелить воду из стакана I в стакан j есть величина Cij.

Помогите Миславу найти порядок, в котором нужно переливать воду из одного стакана в другой такой, чтобы сумма усилий оказалась минимальной.

Ввод

Первая строка ввода содержит целые числа N и K (1<=K<=N<=20).
Следующие N строк содержат по N целых чисел Cij (0<=Cij<=10^5).
i-ая строка и j-ый столбец содержат величину Cij.
Гарантируется Сii=0 для всех i.

Вывод

Выведите минимальное количество усилий, которое требуется Миславу, чтобы достичь цели.

Ввод  
3 3 
0 1 1      
1 0 1 
1 1 0

Вывод
0

Ввод
3 2 
0 1 1      
1 0 1 
1 1 0  

Вывод
1

Ввод
5 2 
0 5 4 3 2 
7 0 4 4 4 
3 3 0 1 2 
4 3 1 0 5 
4 5 5 5 0

Вывод
5

Пояснение к первому тесту:
Миславу не нужно переливать, поскольку он может выпить из 3 стаканов.

Пояснение ко второму тесту
Мислав должен перелить воду ровно из одного стакана (неважно какого) в любой другой,
чтобы осталось два стакана с водой.

Пояснение к третьему тесту:
Для того чтобы получить минимальный ответ 5,
он должен перелить из стакана 4 в стакан 3 (усилие 1), потом 3->5 (усилие 2), и наконец 1->5 (усилие 2).
Суммарное усилие: 1+2+2 = 5.
