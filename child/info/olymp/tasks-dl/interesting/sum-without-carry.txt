Заданы N (1 <= N <= 20) чисел w_1 ... w_N (все в интервале 
от 1...100,000,000). Вам требуется определить размер наибольшего 
подмножества этих чисел, которые можно сложить без переноса при сложении.

Формат ввода

Первая строка содержит N.
Каждая из последующих N строк содержит одно целое число w_i.

Формат вывода

Одно число - максимальное количество чисел, которые
могут быть сложены без переноса.


Пример ввода

5
522
6
84
7311
19

Имеется 5 чисел: 522, 6, 84, 7311, 19.

Пример вывода
3

Пояснение:

Три числа 522, 6, 7311, могут быть сложены без переноса. 

   522
     6
+ 7311
------
  7839
