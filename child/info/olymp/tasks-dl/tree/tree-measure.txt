Задано дерево, имеющее N (3 <= N <= 99999; N нечетное) ребер, длиной 1, последовательно пронумерованных от 1 до N.

По заданной карте всех соединений, сделайте список расстояний от корня до конца каждого ребра.

Карта содержит C (1<= C <= N) соединений, каждое из которых специфицируется тремя целыми числами:
входящее ребро E_i (1 <= E_i <= N)
и два отходящих рёбра B1_i и B2_i (2 <= B1_i <= N; 2 <= B2_i <= N).
Ребро с номером 1 подсоединяется к корню дерева.

Формат ввода

В первой строке содержатся два разделенных пробелом целых числа N C.
Каждая из последующих C строк описывают точку ветвления i разделенными пробелом целыми числами E_i B1_i B2_i

Формат вывода

N строк, строка с номером i содержит одно целое число - расстояние от корня до конца ребра с номером i


Пример ввода

5 2
3 5 4
1 2 3

Пояснения:

Данный ввод  описывает такую карту

                    +------+
                    |Корень|
                    +------+
                       | 1
                       *
                    2 / \ 3
                         *
                      4 / \ 5

Пример вывода

1
2
2
3
3

Пояснения

Ребро 1 всегда на расстоянии 1 от корня дерева, рёбра 2 и 3, присоединенные непосредственно к ребру 1, заканчиваются на расстоянии 2.
Рёбра 4 и 5 исходящие из ребра 3, имеют расстояние 3 от корня.
