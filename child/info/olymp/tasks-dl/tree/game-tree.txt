Требуется защитить средневековый город, дороги которого образуют дерево.
Необходимо поставить минимальное количество стражников в вершины так, чтобы они могли обозревать все дороги.

Формат ввода

Входной файл содержит несколько наборов тестов в текстовом формате.
Каждый тест представляет дерево следующим образом:
- количество вершин
- описание каждой вершины в следующем формате:
номер_вершины:(количество_дорог) номер_вершины1 номер_вершины2 ... номер_вершины_количество_дорог
или
номер_вершины:(0)
Номера вершин - целые числа от 0 до n-1 для n вершин (0 <= n <= 1500).
Каждая дуга появляется во входном тесте ровно один раз.

Например, для дерева:

    0
    |
    1
  /   \ 
 2      3

Решение - один стражник (в вершине 1)

Формат вывода

Вывод для каждого теста должен содержать одно целое число, (каждое в отдельной строке) - минимальное количество стражников для этого теста.

Первый тест

Пример ввода
4
0:(1) 1
1:(2) 2 3
2:(0)
3:(0)
5
3:(3) 1 4 2
1:(1) 0
2:(0)
0:(0)
4:(0)

Пример вывода
1
2
