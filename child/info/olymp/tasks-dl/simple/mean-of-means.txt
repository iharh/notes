64Mb 2sec

Задан двумерный массив N*N (2 <= N <= 99; N нечетное) целых чисел R_ij (1 <= R_ij <= 1,000).
Требуется найти среднее в каждой строке, то есть такое число R_ij в строке,
которое больше либо равно половине чисел этой строки и меньше либо равно половине чисел из этой строки.

Из этих N средних требуется вновь выбрать среднее по тем же правилам.

Формат ввода
* Строка 1: Одно целое число: N
* Строки 2..N+1: Строка i+1 содержит N разделенных пробелами целых чисел в ряду i

Формат вывода
* Строка 1: Одно целое число – среднее из средних

Пример ввода (файл perfect.in):
5
1 5 3 9 5
2 5 3 8 1
6 3 5 9 2
8 8 3 3 2
5 4 4 4 4

Пример вывода (файл perfect.out):
4

Пояснения

N=5, поэтому имеется 25 чисел в матрице 5 x 5.
Средние в строках равны соответственно 5, 3, 5, 3, 4.
Среднее из этих пяти чисел есть 4.
