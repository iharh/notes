64Mb 2sec

Задан список из N слов.
Необходимо для каждого слова определить количество слов, которые совпадают с ним и находятся выше по списку.

Формат ввода:
В первой строк задано одно число N.
В последующих строках заданы слова из маленьких латинских букв.

Ограничения:
1<=N<=100000; длина слов не превосходит 20.

Формат вывода:
N чисел по одному в строке.

Пример ввода:
5
gcjb
am
t
am
am

Пример вывода:
0
0
0
1
2
