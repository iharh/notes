Имеется N (1 <= N <= 10,000) чисел H_i (1 <= H_i <= 1,000,000,000)
Определим "гору" как последовательность чисел H_i, которая сначала возрастает  (или  остается  такой же),
а затем убывает (или остается такой же).

Например:

2, 3, 3, 5, 4, 4, 1.

По краям последовательности, гора может только возрастать или только убывать.

"Шириной горы" называется количество чисел в такой последовательности.

Определите самую широкую гору.

Вот пример
           ooooooo                   o
          ooooooooo                 ooo
          oooooooooo               ooooo
          ooooooooooo           ooooooooo               o
o      ooooooooooooooooo       ooooooooooo             ooo o
oo    ooooooooooooooooooo     ooooooooooooo   o o     ooooooo      o
oooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooo
?\\===//==//======\\\\==\\====////////\\\\\\==/\/\====///\\/\\=====/\=
3211112333677777776543332111112344456765432111212111112343232111111211
aaaaa                     cccccccccccccccccccc eeeeeee    ggggggggg
  bbbbbbbbbbbbbbbbbbbbbbbbbbbb             ddddd ffffffffff  hhhhhhhhh

Горы  помечены  символами  'a',  'b',  и т.д. Очевидно, гора b самая широкая - ее ширина равна 28.

Формат ввода

* Строка 1: Одно целое число: N

* Строки 2..N+1: Строка i+1 содержит одно целое число: H_i

Пример ввода

7
3
2
3
5
4
1
6


Формат вывода

* Строка 1: одно целое число - ширина самой широкой горы.

Пример вывода

5

Пояснения:

Самая широкая гора включает измерения: 2, 3, 5, 4, 1.
Другие горы включают измерения 3, 2 и 1, 6.
