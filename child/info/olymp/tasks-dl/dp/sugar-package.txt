32Mb 2sec

Требуется получить N килограммов сахара, используя два типа пакетов.
Один вмещает 3 килограмма сахара, другой - 5, взяв как можно меньше пакетов.
Например, если нужно получить 18 кг сахара, можно использовать 6 пакетов по 3 кг,
но предпочтительнее использовать три 5-кг пакетов и один 3-кг пакет, взяв всего 4 пакета.

Ввод

Первая и единственная строка ввода содержит целое число N (1<=N<=5000).

Вывод

Первая и единственная строка вывода должна содержать одно число -
минимальное количество пакетов, которое можно использовать
Если невозможно получить ровно N кг - выведите -1.

Примеры

Ввод    Вывод
4       -1

Ввод    Вывод
9       3

Ввод    Вывод
18      4
