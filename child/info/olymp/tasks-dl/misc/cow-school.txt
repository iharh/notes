Бесси собирается в школу.
Она сдала N тестов N (1 <= N <= 5000 кроме одного случая, когда 1 <= N <= 50,000)
и получила T_i баллов из P_i возможных баллов за тест i (0 <= T_i <= P_i < 40,000; 0 < P_i).

Ее учитель отбросил D тестов с самыми низкими процентами (P_i/T_i),
прежде чем вычислить окончательную оценку Бесси
(которая считается как сумма набранных баллов по оставшимся тестам,
деленная на сумму всех баллов, которые можно было набрать в оставшихся тестах).

Бесси сильна в математике и быстро поняла, что такой подход не дает ей максимальной оценки.

Для того, чтобы доказать свою точку зрения, Бесси хочет найти все значения D такие,
для которых она может получить лучшую финальную оценку, отбросив другие тесты (вместо выбранных учителем).
Помогите Бесси найти и вывести все значения D, для которых это возможно.

Необходимо учитывать, что Бесси имеет РАЗЛИЧНЫЕ проценты для всех тестов.

Формат ввода:

* Строка 1: Одно целое число N

* Строки 2..N+1: Строка i+1 содержит два разделенных пробелом целых числа T_i и P_i

Пример ввода (файл schul.in):

5
1 2
5 9
3 8
4 10
1 3

Пояснения к примеру ввода:

Бесси сдала 5 тестов с оценками 1/2, 5/9, 3/8, 4/10, 1/3.

Формат вывода:

* Строка 1: Одно целое число K (0 <= K <= N) - количество таких значений
            D, для которых Бесси может получить лучшую оценку, чем ту,
            которую получил учитель.

* Строки 2..K+1: значения D в порядке возрастания.

Пример вывода (файл schul.out):

2
1
2

Пояснения к примеру вывода:

Для D = 1 отбрасывание 1/3 приведет к результату 13/29.
Более высокий результат 11/24 может быть получен отбрасыванием 3/8.

Для D = 2 отбрасывание 1/3 и 3/8 приведет к результату 10/21.
Более высокий результат 7/14 может быть получен отбрасыванием 3/8 и 4/10.
