64Mb 2sec

По заданному N (1 <= N <= 16) и соответствующим числам W_i, определите максимальную сумму,
которую можно составить из чисел W_i, не превысив указанный предел C (100 <= C <= 5000).

Формат ввода

* Строка 1: Два разделенных пробелом целых числа: C N
* Строки 2..N+1: Строка i+1 содержит одно целое число: W_i
Формат вывода

* Строка 1: Одно целое число - максимальная сумма, не превышающая указанный предел

Пример ввода
259 5
81
58
42
33
61

Пример вывода
242

Пояснения
81+58+42+61 = 242; эта наибольшая возможная сумма
