64Mb 2sec

Имеется N (1 <= N <= 50,000) чисел a[i]  (1 <= a[i] <= 1,000,000).
Имеется список из Q (1 <= Q <= 200,000) непрерывных групп чисел.
Для каждой группы необходимо определить разницу между самым большим и самым маленьким числом в группе.

Формат ввода

* Строка 1: Два разделенных пробелом целых числа N и Q.
* Строка 2..N+1: Строка i+1 содержит одно целое число - a[i].

* Строки N+2..N+Q+1: Два целых числа A и B (1 <= A <= B <= N),
            представляющих группу чисел от A до B включительно.

Формат вывода:

* Строки 1..Q: Каждая строка содержит одно целое число -
               разность между самым большим и самым маленьким числом в данной группе.

Пример ввода (файл lineupg.in):
6 3
1
7
3
4
2
5
1 5
4 6
2 2

Пример вывода (файл lineupg.out):
6
3
0
