Биологи открыли странную DNA-молекулу, описываемую как последовательность из N символов множества {A, B}. Мутация превратила ее в строку, состоящую только из букв {A}. Биологи обнаружили два типа мутаций:
1) Мутация одного символа в другой (A->B или B->A)
2) Мутация целого префикса последовательности (с позиции 1 до позиции K для некоторого K от 1 до N включительно), состоящего из олдних и тех же символов, другим символом (A на B, и B на A).
Вычислите минимальное количество мутаций, которое может конвертировать исходную молекулу в ее конечное состояние (только из символов A) . Мутации могут происходить в произвольном порядке.

Ввод

Первая строка ввода содержит положительное целое число N (1 ≤ N ≤ 1 000 000), длину молекулы. Вторая строка ввода содержит строку из N символов, каждый из которых A или B. Эта строка представляет исходное состояние молекулы.

Вывод

Первая и единственная строка вывода должна содержать требуемое минимальное количество мутаций.

Примеры
input 
4 
ABBA

output 
2

input 
5 
BBABB

output 
2

input 
12 
AAABBBAAABBB

output 
4
