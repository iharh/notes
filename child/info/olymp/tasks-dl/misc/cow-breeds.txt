N коров Фермера Джона, последовательно пронумерованных от 1…N, выстроены в ряд.
Каждая корова имеет ID породы:
    1 - Holsteins, 2 - Guernseys, 3 - Jerseys.

ФД просит Вас посчитать количество коров каждой породы, внутри некоторого интервала этого порядка.

ФОРМАТ ВВОДА (файл bcount.in):
Первая строка ввода содержит N и Q (1≤N≤100,000, 1≤Q≤100,000).
Следующие N строк содержат целое число 1,2, или 3 - ID породы соответствующей коровы.

Следующиеt Q строк описывают запрос в виде двух целых чисел a,b (a≤b).

ФОРМАТ ВЫВОДА (файл bcount.out):
Для каждого из Q запросов (a,b), выведите строку, содержащую три целых числа количество коров в интервале a…b, имеющих номера пород 1,2,3.

ПРИМЕР ВВОДА:
6 3
2
1
1
3
2
1
1 6
3 3
2 4

ПРИМЕР ВЫВОДА:
3 2 1
1 0 0
2 0 1
