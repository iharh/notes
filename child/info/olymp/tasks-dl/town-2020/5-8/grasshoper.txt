Имеется решетка со свободными и занятыми квадратиками. Кузнечик может прыгать по свободным квадратикам, перепрыгивая через занятые.

Его цель проскакать от старта до финиша за минимальное количество прыжков. Кузнечик начинает прыгать в формате (0,0). Перед прыжком он может любое из этих двух чисел увеличить на 1, уменьшить на 1 и оставить без изменения. Например, если текущий формат (2,1), то на очередном ходу кузнечик может прыгнуть любым из следующих способов: (1,0), (1,1), (1,2), (2,0), (2,1), (2,2), (3,0), (3,1) , (3,2).

Имеется также ограничение - оба числа формата прыжка должны оставаться в диапазоне от -3 до 3 включительно.

Формат ввода

Первая строка содержит количество тестов (N). Каждый тест имеет следующий формат: Первая строка содержит ширину X (1 <= X <= 16) и высоту Y (1 <= Y <= 16) решетки. Следующая строка содержит 4 целых числа, разделенных одиночными пробелами, которые указывают стартовую (x1,y1) и конечную (x2,y2) клетки (0 <= x1 < X, 0 <= x2 < X, 0 <= y1 < Y, 0 <= y2 < Y). Третья строка содержит целое число P указывающая количество препятствий на решетке. Далее следуют P строк, каждая из которых указывает препятствие. Описание каждого препятствия состоит из 4 чисел x1 x2 y1 y2 (0 <= x1 <= x2 < X, 0 <= y1 <= y2 < Y), означающее, что все квадраты (x,y) c x1 <= x <= x2 и y1 <= y <= y2, заняты. Гарантируется, что стартовая и конечная точка не будут заняты.

Формат вывода

Выведите
No solution.
если нет пути от старта до финиша оп описанным правилам при заданных препятствиях
Иначе выведите
Optimal solution takes N hops.
где N - количество прыжков от старта до финиша
Пример ввода                 
2
5 5
4 0 4 4
1
1 4 2 3
3 3
0 0 2 2
2
1 1 0 2
0 2 1 1
Пример вывода
Optimal solution takes 7 hops.
No solution.
