Удивительное число - это число, которое может быть разбито на два простых числа, каждое из которых имеет не менее D цифр и, конечно, не начинается с 0. Например, если D=2, то число 11329 - это удивительное число, поскольку его можно представить составленным из соединения чисел 113 и 29, каждое из которых - простое.

Ваша задача - найти первое удивительное число, большее либо равное заданного целого числа N (1 <= N <= 2,000,000,000). Ответ тоже не превысит 2,000,000,000.

Формат ввода

* Строка 1: Два разделенных пробелом целых числа D и N

Формат вывода

* Строка 1: Строка, которая содержит первое удивительное число не меньшее чем N.

Пример ввода (файл wpb.in):
2 11328

Пример вывода (файл wpb.out):
11329
