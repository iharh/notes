Имеется число из N цифр. Требуется получить наибольшее число,
удалив ровно K из них.

Ввод

Первая строка ввода содержит целые числа N и K (1 <= K < N <= 500 000).
Следующая строка содержит число из N цифр. Число начинается с ненулевой цифры.

Вывод

Первая и единственная строка вывода должна содержать наибольшее число,
которое можно получить удалением ровно K цифр из данного числа.

Примеры:

Ввод
4 2
1924

Вывод
94

Ввод
7 3
1231234

Вывод
3234

Ввод
10 4
4177252841

Вывод
775841
