В этот пасьянс играют используя N x N (3 <= N <= 7) решетку обычных игральных карт четырех мастей по-английски называемых
    Clubs, Diamonds, Hearts, Spaces
и 13 карт, по-анлийски называемых
    Ace, 2, 3, 4, ..., 10, Jack, Queen, King.

Карты обозначаются двумя символами -
    первый символ - значение: A, 2, 3, 4, ..., 9, T, J, Q, K
    второй символ - масть   : C, D, H, S.

Ниже представлен пример начальной позиции карт для N=4:
     8S AD 3C AC     (8 Spades, Ace Diamonds ...)
     8C 4H QD QS
     5D 9H KC 7H
     TC QC AS 2D

Игра начинается с левого нижнего угла, делается ровно ровно 2*N-2 ходов "вправо" или "вверх" - к правому верхнему углу.
По пути, игрок собирает очки с каждой карты
    (Ace(туз) - 1 очко, 2 - 2 очка, ..., 9 - 9 очков, T - 10 очков, J - 11, Q - 12, K - 13)
которую она проходит.

Цель игрока - собрать как можно больше очков.

Например, если идти по пути:
    TC-QC-AS-2C-7H-QS-AC
то счет будет
    10+12+1+2+7+12+1=45.

А если пойти по такому пути:
    TC-5D-8C-8S-AD-3C-AC
то счет будет
    10+5+8+8+1+3+1=36.
что хуже (меньше) чем в первом случае.

Максимальный счет для данной раскладки - 69 :
    маршрут TC-QC-9H-KC-QD-QS-AC
    сумма = 10+12+9+13+12+12+1).

Требуется находить максимальную сумму.

Формат ввода:

* Строка 1: Одно целое число: N
* Строки 2..N+1: Строка i+1 содержит описание карт в строке i
                 (строка 1 - верхняя строка) используя N
                 разделенных пробелом описаний карт.

Формат вывода:

* Строка 1: Одна строка, содержащая одно целое число - наилучший
            (максимальный) счет, который может быть достигнут.


Пример ввода
4
8S AD 3C AC
8C 4H QD QS
5D 9H KC 7H
TC QC AS 2D

Пример вывода
69
