Azure Managed Identity — это функция в облаке Microsoft Azure, которая позволяет автоматически управлять учетными данными для приложений и сервисов, 
чтобы они могли безопасно получать доступ к другим ресурсам Azure без необходимости вручную хранить или управлять паролями, секретами или ключами. 
Managed Identity интегрируется с Azure Active Directory (AAD) и предоставляет приложению уникальную идентификацию, 
которую можно использовать для аутентификации и авторизации при доступе к ресурсам, 
таким как Azure Key Vault, Azure Storage, SQL Database и другим сервисам, поддерживающим AAD-аутентификацию.

Существует два типа Managed Identity:
* System-assigned Managed Identity — автоматически создается и управляется Azure для конкретного ресурса 
  (например, виртуальной машины или веб-приложения). Удаляется вместе с ресурсом.
* User-assigned Managed Identity — создается вручную и может быть назначена одному или нескольким ресурсам.
