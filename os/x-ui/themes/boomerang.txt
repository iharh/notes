Extract the theme file you have downloaded to
    ~/.themes

If the folder doesn't exist, create it.
Open
    lxappearance
and select the desired theme.
To customize the Window borders, type
    obconf
    obconf-qt
in a terminal.
Choose the desired Openbox theme in the list and exit the app.
